在webStorm中运行ts，修复--require ts-node/register问题

如果我们想要在webStorm中像运行js文件那样运行ts文件，我们可以：

  1. 全局安装ts：npm install -g typescript
  2. 全局安装ts-node：npm install -g ts-node
  3. 安装插件：File-Settings-Plugins，搜索Run Configuration for TypeScript 如果你在ts文件中右键没有出现运行代码的选项，尝试重启IDE
  4. 配置插件：如果你运行代码显示错误"<node_path>/node"--require ts-node/register错误，那么请你点击上方菜单栏的 run-Edit Configurations，选择ts-node路径

如果你安装ts的时候除了全局安装还执行了全局安装，可能不会出现第四步。是否需要本地安装可以根据您的喜好自行决定。
